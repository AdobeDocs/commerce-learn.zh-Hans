# 了解Salesforce Commerce Cloud入门套件架构

了解如何使用Adobe Commerce Optimizer设置和部署适用于Salesforce Commerce Cloud的App Builder Starter Kit。 了解如何在Adobe Developer Console中创建新项目、添加必要的API以及下载工作区JSON文件。 了解克隆starter kit存储库、设置配置文件以及安装Adobe IO CLI以将本地代码库与开发控制台项目链接的过程。 了解填写环境变量和同步站点详细信息，以及验证应用程序是否已成功构建和部署的流程。 了解一些载入步骤（如注册操作），并在开发控制台工作区中确认部署。

## 此视频面向谁？

* Commerce解决方案架构师
* 技术营销工程师
* 电子商务平台管理员

## 视频内容

* 设置Adobe Developer Console项目并添加Commerce Optimizer集成所需的API。
* 克隆App Builder入门工具包，并使用Adobe IO CLI配置环境文件。
* 部署应用程序，并确认在您的工作区中成功注册操作和事件。
