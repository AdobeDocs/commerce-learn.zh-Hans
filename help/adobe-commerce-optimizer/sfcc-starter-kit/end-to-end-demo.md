# 观看Salesforce Commerce Cloud入门工具包的端到端演示

一个端到端演示，其中演示了如何使用入门工具包，在Salesforce Commerce和Commerce Optimizer之间成功设置和执行完全的差异同步。 了解如何通过运行时日志和事件跟踪确认同步是否成功，然后在实时店面中验证结果。 了解差异同步功能的工作原理。 其中还包括修改产品详细信息和在SFC目录中进行定价等步骤，系统会自动在Commerce Optimizer中检测和更新这些更改，并立即在店面中反映这些更改。

## 此视频面向谁？

* Commerce解决方案架构师
* 技术营销工程师
* 电子商务平台管理员

## 视频内容

* 使用App Builder和入门工具包，在Salesforce Commerce Cloud和Commerce Optimizer之间设置和执行完全同步。
* 启动完全同步，并通过运行时日志和事件跟踪确认成功。
* 在Commerce Optimizer中，可通过Delta Sync过程自动检测和更新SFC目录中的产品详细信息和定价变更。
* 实时更新和演示系统之间的无缝数据流和同步。
