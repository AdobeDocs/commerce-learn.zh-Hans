# 安装SFCC墨盒

了解如何为Salesforce Commerce Cloud实例安装和配置Adobe Commerce Optimizer墨盒。 此视频介绍如何克隆代码存储库、使用VS Code和Prophet扩展设置墨盒以及激活新代码版本。 它说明了如何为站点启用墨盒、配置增量导出以及导入自定义对象和作业定义。 了解如何验证作业设置并测试自定义API端点，以确认安装并同步您的目录。


## 此视频面向谁？

* Commerce解决方案架构师
* 技术营销工程师
* 电子商务平台管理员

## 视频内容

* 使用VS代码和Prophet扩展安装和配置Adobe Commerce Optimizer墨盒。
* 在Business Manager中创建并激活新的代码版本，并为特定站点启用盒式磁带。
* 设置Delta Exports以管理目录和价格手册更新。
* 导入自定义对象和作业定义以支持跟踪和同步。
* 测试自定义API端点以确认安装成功并准备目录同步。
