# Commerce MCP概述

[!BADGE 即将推出]{type=Informative tooltip="此功能即将推出，并且可能会发生更改。"}

适用于Adobe Commerce的Commerce MCP可帮助开发人员使用实时商务数据和工具构建从发现到检验的自定义代理体验。 在此视频中，自定义AI助手使用Commerce MCP通过了解购物者意图并提供相关产品推荐将对话转换为转化。

## 此视频面向谁？

* Commerce商店所有者
* 促销商
* 数字营销人员

## 视频内容

* 使用自然语言而不是筛选器和搜索栏进行对话式产品发现
* 上下文感知会话管理保留购物车内容和购物者偏好设置
* 通过键入进行双向购物车管理，无需重新加载页面或确认模式
* 结帐已在对话中完成，未导航离开
* 特定于所购买产品的购买后上下文指南
