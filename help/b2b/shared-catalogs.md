# 管理共享目录

{{b2b-feature}}

Adobe Commerce的B2B使您能够使用自定义定价为不同的公司维护封闭共享目录。 除了标准的主要产品目录外，它还允许客户访问具有不同定价结构的两种共享目录。

如果配置中启用了共享目录功能，则管理员仍可看到原始主目录，但店面中仅可看到默认（常规）公共共享目录。 此外，可以创建仅对特定公司帐户成员可见的自定义目录。
