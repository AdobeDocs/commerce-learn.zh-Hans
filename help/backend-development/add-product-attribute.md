# 创建产品属性

添加产品属性是中最常用的操作之一 [!DNL Commerce]. 属性是解决许多与产品相关的实际任务的强大方法。 向产品添加下拉列表类型属性的过程非常简单。

在本视频中：

- 添加一个名为clothing_material的属性，可能的值包括：棉花、皮革、丝绸、牛仔布、毛皮和羊毛
- 以粗体文本使此属性在产品视图页面上可见
- 将其分配给Default属性集并添加限制
- 添加新属性
