# 创建模块

模块是结构元素 [!DNL Commerce]  — 整个系统基于模块构建。 通常，创建自定义项的第一步是构建模块。

## 此视频面向谁？

- 开发人员

## 添加模块的步骤

- 创建模块文件夹。
- 创建etc/module.xml文件。
- 创建registration.php文件。
- 运行bin/magento设置。
- 升级脚本以安装新模块。
- 检查模块是否正常工作。
