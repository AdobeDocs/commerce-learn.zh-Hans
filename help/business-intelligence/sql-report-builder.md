# MBI - SQLReport Builder

SQLReport Builder允许您直接查询data warehouse、查看结果，并快速将它们转换为图表。 使用SQL构建报告的最好部分是，您不必等待更新周期，即可在您创建的列上进行迭代。 如果结果不正确，您可以快速编辑并重新运行查询，直到其结果与您的预期相符。

>[!IMPORTANT]
>
>您必须具有管理员权限才能访问SQLReport Builder。

在此视频中：

- 运行SQL查询
- 可视化结果
- 将报表保存到功能板
