# SQL REPORT BUILDER

Adobe Commerce Intelligence SQL Report Builder允许您查询数据仓库、查看结果并将它们转换为图表。 使用SQL构建报表时，不必等待更新周期来优化您创建的列。 如果出现问题，请编辑并重新运行查询，直到输出与您预期的结果匹配。

>[!IMPORTANT]
>
> 要使用SQL Report Builder，您需要管理员权限。

>[!TIP]
>
> 如果您不熟悉Commerce Intelligence，[Adobe Commerce Intelligence用户指南](https://experienceleague.adobe.com/docs/commerce-business-intelligence/mbi/guide-overview.html){target="_blank"}将介绍如何聚合、分析和可视化数据以获得更强大的业务洞察。

## 此视频面向谁？

* 商业用户
* 数据分析师

## 视频内容

* 打开SQL Report Builder并对您的数据仓库运行查询。
* 从查询结果构建可视化。
* 将报告保存到团队的仪表板。
