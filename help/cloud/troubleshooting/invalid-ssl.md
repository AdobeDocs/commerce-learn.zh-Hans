# 已过期的SSL证书

了解如何识别过期的ssl证书，以及如何为Adobe Commerce Cloud集成或其他开发环境重新生成一个证书。

## 此视频面向谁

* 开发人员使用Adobe Commerce Cloud集成和其他开发人员分支进行测试。
* 使用Luma店面测试功能的新Adobe Commerce Cloud队友。

## 视频内容

* 了解如何检测无效的ssl证书。
* 识别在此过程中何时创建新证书。
* 验证新证书是否按预期工作。
