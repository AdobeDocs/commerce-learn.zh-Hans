# Adobe Commerce as a Cloud Service概述

Adobe Commerce as a Cloud Service为企业提供了灵活、可扩展且高效的解决方案，具备弹性自动缩放和自动更新等云原生功能。 它通过自助服务配置简化了入职流程，并拥有可组合的架构，以实现独立服务部署和API优先扩展。 预配置服务和持续提供的新功能可确保最佳性能和易用性，使其成为动态成长型企业的理想之选
