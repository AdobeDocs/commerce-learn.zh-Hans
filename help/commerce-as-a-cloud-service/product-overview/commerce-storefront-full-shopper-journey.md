# 提供端到端购物之旅

了解Adobe Commerce as a Cloud Service如何提供从产品发现到结账和购买后支持的无缝购物体验。 了解实时搜索、产品推荐和自助功能板等内置工具如何提高转化率并减少运营开销。

## 此视频面向谁？

* 电子商务管理器
* 客户体验专家
* 数字营销经理

## 视频内容

* 了解如何使用AI支持的实时搜索和产品推荐使购物体验个性化。
* 了解自助式帐户工具如何简化客户的订单跟踪、退货和重新订购。
