# 添加JavaScript模块

{{only-for-on-prem-commerce-cloud}}

了解如何开发一个简单的JavaScript模块，在店面中显示问候语“Hello World”。

## 此视频面向谁？

* 开发人员

## 步骤

1. 创建模块。
2. 添加`requirejs-config.js`和JavaScript模块文件。
3. 创建布局更新以加载模块的模板。
4. 创建模板文件。
5. 在店面注册模块并进行测试。

## 视频内容

* 设置模块并要求JS配置。
* 通过布局更新和模板连接脚本。
* 验证页面上的问候语。
