# 创建新页面

{{only-for-on-prem-commerce-cloud}}

创建一个返回带有一个参数的json的页面。

## 此视频面向谁？

* 开发人员

## 添加页面的步骤

* 创建模块
* 添加&#x200B;**routes.xml**&#x200B;文件
* 添加控制器（操作）文件

## 创建模块的步骤

* 创建模块文件夹
* 创建`etc/module.xml`文件
* 创建`registration.php`文件
* 运行`bin/magento setup:upgrade`脚本以安装新模块
* 检查模块是否正常工作
