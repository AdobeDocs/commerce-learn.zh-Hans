# 创建产品属性

Adobe Commerce使用属性来管理与产品、客户和产品评级相关的信息。 属性是可能在表单、报表、产品评论、分层导航和电子邮件消息中显示为字段的信息段。 它们是产品目录的构建基块，用于描述产品的特定特性。
