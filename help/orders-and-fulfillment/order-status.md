# 管理订单状态

客户下达订单时，销售订单将创建为事务处理的临时记录。 在收到付款之前，销售订单的状态为&#x200B;_待处理_。 处于“待定”状态时，可以编辑或取消订单，直到收到付款并生成发票为止。 在本视频中，了解如何检查订单状态及其详细信息，以及如何根据需要更改订单的状态。
