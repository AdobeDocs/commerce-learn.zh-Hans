# 创建静态块和动态块

块是内容的模块化单位，可放置在页面上的大多数位置。 内容块有时称为静态块或CMS块。 它们可用于显示固定信息，例如文本、图像和嵌入视频，以及由小组件提供的或源自数据库或其他源中的动态信息。 主页上的大多数元素都是可以轻松管理的块。 在本视频中，了解如何创建静态块和动态块。
