# 云自动化修补服务

Adobe的Cloud Automation Patching Service (CAPS)是Adobe Commerce站点范围分析工具(SWAT)中的一项功能，可简化发现和应用安全修补程序的过程。 用户通过SWAT功能板下的Commerce管理员访问CAPS，选择其项目和环境，并查看可用的修补程序。

## 此视频面向谁？

* 网站管理员
* 开发人员
* DevOps和系统管理员
