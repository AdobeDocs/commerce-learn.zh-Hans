# 使用Quality Patch工具

Quality Patch Tool是一个命令行工具，可为Adobe Commerce和Magento Open Source提供高质量的修补程序。 它允许您：

- 查看有关最新质量修补程序的常规信息
- 应用质量补丁程序
- 还原应用的修补程序

Adobe Commerce支持团队和Magento Open Source社区将开发质量补丁。

## 此视频面向谁？

- 后端开发人员
- 站点管理器
