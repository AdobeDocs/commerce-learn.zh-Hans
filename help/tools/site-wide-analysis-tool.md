# 站点范围分析工具仪表板

全站点分析工具是一种主动式自助服务工具和中央存储库，其中包含详细的系统分析和建议，以确保Adobe Commerce安装的安全性和可操作性。 它提供全天候的实时性能监控、报告和建议，以发现潜在问题。 使用此工具可以更好地了解站点运行状况、安全性和应用程序配置，有助于缩短解决时间，并改善站点稳定性和性能。
