# 站点范围分析工具功能板

站点范围分析工具是一个主动的自助服务工具和中央存储库，包括详细的系统分析和建议，可确保Adobe Commerce安装的安全性和可操作性。 它提供24/7实时性能监控、报告和建议以确定潜在问题。 使用此工具可更好地了解站点运行状况、安全性和应用程序配置，帮助缩短解析时间，并提高站点稳定性和性能。
