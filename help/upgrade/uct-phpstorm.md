# 在PHP Storm上使用升级兼容工具

升级兼容性工具(UCT)是一款免费工具，可在几分钟内分析当前版本与目标升级版本之间的不兼容性。 PHP Storm插件使此工具更易于使用。

在此视频中：

- 什么是升级兼容性工具？
- 如何将其与MagentoPHP Storm插件结合使用
- 如何访问HTML结果，以帮助按不兼容性的严重性和数量显示模块
