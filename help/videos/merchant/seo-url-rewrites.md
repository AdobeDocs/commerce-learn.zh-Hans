# 为SEO管理URL重写

“URL重写”工具允许您更改与产品、类别或CMS页面关联的任何URL。 重写生效后，指向之前URL的任何链接都将被重定向到新地址。

## 这个视频给谁？

- 网站管理器
- 电子商务促销商
- 电子商务营销人员
