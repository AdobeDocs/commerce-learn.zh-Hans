# 创建静态块和动态块

块是内容的模块化单元，可放置在页面上的最大位置。 内容块有时称为静态块或CMS块。 它们可用于显示固定信息，如文本、图像和嵌入的视频，以及由小组件提供或源自数据库或其他源的动态信息。 主页上的大多数元素都是可轻松管理的块。 在此视频中，了解如何创建静态块和动态块。
